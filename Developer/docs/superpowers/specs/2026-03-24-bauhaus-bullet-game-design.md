## 一、项目概述

一款运行于浏览器的弹幕躲避游戏，融合包豪斯几何美学与实验性合成器音乐。玩家点击控制一个圆形角色在弹幕中穿行，游戏以"常规→狂暴"的节奏循环驱动张力。视觉语言基于康定斯基的基本形-色彩理论（点线面 × 三原色），音乐与弹幕预先编排同步，形成音乐通感体验。

---

## 二、核心玩法循环

### 操控方式
- 点击场地任意位置，玩家圆立即朝该点匀速移动
- 无目标点标记，操作界面干净直接
- 可随时点击新目标点覆盖当前移动方向

### 血量系统
- 玩家拥有 **4 格 HP**，UI 以 4 个小矩形（Quadrat）显示
- 被弹幕击中：-1 HP，触发 1.5 秒无敌帧（角色半透明闪烁）
- 狂暴模式结束后：HP **直接设为 2 格**（无论当前 HP 是多少，一律重置为 2）
- HP 归零：游戏结束，展示存活时长与最终得分

### 节奏循环
```
[ 常规期 30s ] → [ 狂暴期 12s ] → [ 常规期 30s ] → ∞
```

**常规期**
- 标准移速
- 弹幕跟随预编排时间轴有序生成
- 背景色：黑色 #1a1a1a

**狂暴期**
- 玩家移速自动提升 ×1.6
- 弹幕速度 ×1.4，密度 ×2
- 追踪三角弹每 3 秒生成一枚（仅狂暴期出现）
- 背景色渐变至深红 #8B0000
- 过渡动画：弹幕形态"解体重组"，冲击波扩散效果

### 得分机制
- 存活每秒：+10 分
- 狂暴期每秒：+30 分（替代常规得分，非叠加）
- HP 满格（4格）时：当前每秒得分 ×1.2（含狂暴期：30 × 1.2 = 36/s）

## 三、视觉设计系统

### 包豪斯色-形对应体系

| 元素 | 颜色 | 几何形 | 包豪斯含义 |
|---|---|---|---|
| 玩家 | 白／米白 #F5F0E8 | 圆 Kreis | 自由、精神、运动 |
| 点型弹幕 | 黄 #FFD600 | 圆点 Punkt | 鼓点节拍、原初张力 |
| 线型弹幕 | 红 #D40000 | 线段 Linie | 低频扫射、方向力 |
| 面型弹幕 | 蓝 #0048A0 | 矩形 Fläche | 和弦扩散、静态稳定 |
| 追踪弹（狂暴专属） | 白 #F5F0E8 描边红 #D40000 | 三角 Dreieck | 张力、追逐、危险 |
| 背景（常规） | 黑 #1a1a1a | — | 静默底色 |
| 背景（狂暴） | 深红 #8B0000 | — | 危险、激烈 |

### HP 显示
- 4 个小矩形排列于画面左上角
- 满格：白色填充
- 扣血：变为黑框空心矩形
- 恢复：对应方块亮起黄色闪光

### 视觉反馈事件
| 事件 | 反馈 |
|---|---|
| 玩家被击中 | 圆圈白色闪烁，1.5s 半透明无敌帧 |
| HP 扣减 | 对应方块变空心 + 低频短促音效 |
| 狂暴开始 | 背景渐变深红，弹幕尺寸瞬间放大再还原（冲击波） |
| 狂暴结束 | 背景渐回黑色，HP 回复方块黄色闪光 |
| 游戏结束 | 所有弹幕静止，得分与时长居中展示 |

### 网格背景
全场维持细密网格线（opacity 约 4%），增强包豪斯构成感，不干扰游戏内容。

## 四、音频设计系统

### 方向
预编排（方向 A）：手工编排三段包豪斯实验性合成器乐章，使用 Tone.js 合成。每段音乐内嵌弹幕时间轴，精确到毫秒同步触发。

### 三段乐章

**乐章一：脉冲 Puls**
- 风格：低频脉冲 + 稀疏打击
- 弹幕主题：黄色圆点为主，环形向外爆发，节拍感强，间隔规律
- 情绪：稳定、律动、初入状态

**乐章二：线条 Linie**
- 风格：旋律线条 + 节拍密集
- 弹幕主题：红色线段为主，旋转扫射与横向平推交替，形成"扫描"感
- 情绪：紧张、流动、考验走位

**乐章三：混沌 Chaos**
- 风格：和声叠加 + 混沌节奏
- 弹幕主题：三种形态混合，节奏不规律
- 情绪：高压、复杂、考验反应

三段乐章各约 2 分钟，无缝循环播放。

### 音乐通感色彩映射
| 音乐元素 | 视觉响应 |
|---|---|
| 鼓点 Beat | 黄色圆点脉冲同步扩散 |
| 低频 Bass | 红色弹幕生成，背景轻微颤动 |
| 和弦 Harmony | 蓝色矩形以扩散阵形生成 |
| 静止 Rest | 弹幕密度归零，留白 |

### 弹幕时间轴数据格式
```js
// patterns.js 示例
{ t: 4200, type: 'dot',  origin: 'top',    count: 5, spread: 'fan',    color: '#FFD600' }
{ t: 8800, type: 'line', origin: 'center', angle: 0, rotate: true,     color: '#D40000' }
{ t: 12000, type: 'rect', origin: 'left',  count: 2, speed: 'slow',    color: '#0048A0' }
```

## 五、技术架构

### 技术栈
- HTML5 Canvas + 原生 JavaScript（无框架）
- **Tone.js** — 合成器音乐生成与弹幕节拍调度
- 浏览器直接打开运行，无需构建工具

### 文件结构
```
game/
├── index.html      # 入口，Canvas 画布，HUD 层
├── main.js         # 游戏主循环（requestAnimationFrame）
├── player.js       # 玩家圆：位置、速度、HP、无敌帧逻辑
├── bullets.js      # 弹幕对象池：生成、移动、碰撞检测
├── patterns.js     # 弹幕图案库：三段乐章的时间轴数据
├── phase.js        # 阶段控制：常规/狂暴切换、计时、过渡动画
├── audio.js        # Tone.js：三段乐章合成器定义、节拍触发
├── renderer.js     # Canvas 绘制：背景、网格、弹幕、玩家、HUD
└── score.js        # 得分计算、倍率、最终结算
```

### 游戏主循环（每帧顺序）
1. 更新阶段计时（phase.js）
2. 触发弹幕时间轴（patterns.js）
3. 更新所有弹幕位置（bullets.js）
4. 碰撞检测（bullets.js × player.js）
5. 更新玩家位置（player.js）
6. 绘制全帧（renderer.js）
7. 更新得分（score.js）

### 游戏参数

| 参数 | 值 |
|---|---|
| 画布尺寸 | 浏览器全窗口（window.innerWidth × window.innerHeight） |
| 玩家视觉半径 | 12px |
| 玩家碰撞半径 | 8px（视觉半径 × 0.7，向下取整） |
| 玩家常规移速 | 220 px/s |
| 玩家狂暴移速 | 352 px/s（× 1.6） |
| 点弹速度（常规） | 180 px/s |
| 线弹速度（常规） | 140 px/s |
| 面弹速度（常规） | 80 px/s |
| 三角追踪弹速度（狂暴） | 160 px/s，固定速度，生成时朝玩家当前位置直线飞行（无追踪转向），继承狂暴 ×1.4 倍速 = 224 px/s |
| 狂暴弹幕速度倍率 | ×1.4 |
| 三角弹生成间隔 | 每 3 秒一枚（从画面随机边缘生成） |
| 无敌帧时长 | 1.5s |

### 乐章与阶段循环的关系

三段乐章独立于狂暴循环运行：乐章按顺序播放（一→二→三→一→…），每段约 2 分钟；狂暴循环（30s/12s）在乐章内部独立计时，二者不互相重置。乐章切换时弹幕时间轴重置到该乐章起点，狂暴相位保持连续。

### 音频与弹幕同步机制

`audio.js` 使用 Tone.js 的 `Transport` 时钟作为全局时间源。`patterns.js` 存储每段乐章的弹幕事件数组（以毫秒为键），`main.js` 每帧读取 `Tone.Transport.seconds`，触发所有 `t <= 当前时间` 且未触发的事件，调用 `bullets.js` 生成对应弹幕。

### 游戏结束界面
弹幕全部停止运动，画面保持当前帧，中央覆盖半透明黑色遮罩，显示：存活时间、最终得分、"CLICK TO RESTART"提示。点击任意位置重新开始游戏（重置所有状态，音乐从乐章一重新播放）。

## 六、弹幕图案系统

### 四种基础形态

| 形态 | 颜色 | 行为描述 | 常规速度 | 出现阶段 |
|---|---|---|---|---|
| 点 Punkt | 黄 #FFD600 | 圆点，单发/扇形/环形扩散，匀速直线飞行，从画面边缘或中心生成 | 180 px/s | 常规 + 狂暴 |
| 线 Linie | 红 #D40000 | 线段（宽 3px，长 32px），旋转扫射或平行推进，从边缘生成 | 140 px/s | 常规 + 狂暴 |
| 面 Fläche | 蓝 #0048A0 | 矩形（14×14px），慢速大型弹，从边缘生成，覆盖区域 | 80 px/s | 常规 + 狂暴 |
| 三角 Dreieck | 白描红边 | 追踪弹，生成时朝玩家当前位置直线飞行（无转向），从随机边缘生成 | 224 px/s（含狂暴倍率） | **仅狂暴** |

### 狂暴模式弹幕变化
- 所有弹幕速度 ×1.4
- 时间轴间隔减半（密度 ×2）
- 额外每 3 秒追加一枚三角追踪弹
